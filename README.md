# 概要
音声編集をUIを含めて解説しています。
実装の流れは、音声を収録して、その収録ファイルを編集しています。
UIの部分は波形の表示の仕方、時間表記の方法などを解説しています。

シンプルに実装するために
収録はAVAudioRecorder、
再生はAVAudioPlayer、
編集はAVAssetExportSessionを用いて実装しています。
